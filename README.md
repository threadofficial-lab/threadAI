# threadAI
ThreadAI is a powerful data intelligence agent designed to transform raw data into clear insights, interactive visualizations, and professional-grade reports. It performs advanced data analysis, statistical modeling, and chart generation across multiple formats including CSV, JSON, spreadsheets, and SQL databases.
### Example Prompts
- “Analyze this dataset and summarize key trends”
- “Create charts showing month-over-month growth”
- “Detect anomalies and explain possible causes”
- “Generate a professional report for stakeholders”
